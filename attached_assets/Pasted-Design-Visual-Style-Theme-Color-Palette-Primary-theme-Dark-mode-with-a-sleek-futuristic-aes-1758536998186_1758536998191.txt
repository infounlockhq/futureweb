Design & Visual Style

Theme & Color Palette:

Primary theme: Dark mode with a sleek, futuristic aesthetic.

Backgrounds: Deep navy (#0a0f1c) or true black (#000000) base, overlaid with layered soft blue and purple gradients (e.g., #4f46e5, #9333ea, #06b6d4).

Gradients should have smooth, flowing transitions that convey motion and digital depth.

Animation Effects:

Particle Effects (via particles.js or react-particles-js):

Use subtle floating particles in the Hero and Services sections.

Colors: Low-opacity white or soft blue.

Behavior: Slow, fluid movement for a calm, tech-savvy atmosphere.

Reactbits.dev Text Animations:

Apply dynamic animations such as:

Shiny Text for key headings.

Split Text or Type Text for CTAs and subheadings.

Rotating Text for showcasing different AI agent capabilities.

Should feel high-tech and premium, not gimmicky.

Typography:

Font: Modern sans-serif (e.g., Inter, Poppins, or Space Grotesk).

Color: White or light pastel tones (e.g., #f0f0ff, #cbd5e1) for contrast.

Line height and spacing optimized for readability across devices.

Background Details:

Grid Line Background Section:

Implement a subtle SVG or CSS linear-gradient grid pattern.

Purpose: To convey technical structure, precision, and innovation.

Use sparingly in a dedicated section to create contrast with flowing gradients.
